# 学生选课系统功能模块说明

## 1. 系统简介

学生选课系统是一个基于 Spring Boot 3、MyBatis-Plus、MySQL 和 Vue 3 构建的教学管理平台，面向管理员、教师和学生三类用户使用。系统围绕课程管理、学生选课、成绩录入和信息维护等核心业务展开，支持基础数据管理与学期选课流程控制。

## 2. 用户角色

### 2.1 管理员

管理员负责系统级维护和教学业务管理，拥有较高权限，可以管理课程、教师、学生、选课记录、选课时间窗口、通知记录和审计日志等内容。

### 2.2 教师

教师主要使用系统维护本人课程、查看相关学生信息，并对学生选课记录进行成绩录入。

### 2.3 学生

学生可以注册和登录系统，查看课程信息、进行选课或退课、查看成绩与 GPA，并维护个人资料。

## 3. 主要功能模块

### 3.1 用户认证模块

- 提供管理员、教师、学生三类用户登录功能。
- 支持学生自助注册。
- 使用加密密码进行身份校验，保证账户安全。

### 3.2 个人资料模块

- 不同角色均可查看和修改本人的基础信息。
- 支持修改账号、密码以及角色对应的个人资料字段。
- 更新后可在系统内即时生效。

### 3.3 管理员管理模块

- 用于维护系统管理员账号信息。
- 支持新增、查询、修改、删除管理员数据。
- 适合后台账号统一管理。

### 3.4 教师信息管理模块

- 维护教师账号、教工号、姓名、职称、电话、性别等信息。
- 管理员可统一维护教师档案。
- 教师也可在个人中心维护本人的资料。

### 3.5 学生信息管理模块

- 管理学生账号、学号、学院、专业、班级、联系方式等信息。
- 支持学生信息的增删改查。
- 为选课与成绩管理提供基础数据支撑。

### 3.6 课程管理模块

- 管理课程名称、课程编号、开课学院、学分、容量、授课教师和上课时间等内容。
- 支持课程新增、编辑、删除与查询。
- 可按学院、教师、学分等条件筛选课程。

### 3.7 选课管理模块

- 学生可在开放时间内进行选课或退课操作。
- 系统会校验重复选课、课程容量和时间冲突等规则。
- 管理员可以查看和维护全局选课记录。

### 3.8 成绩管理模块

- 教师可对本人课程对应的学生选课记录录入成绩。
- 学生可查看自己的成绩信息和 GPA 统计结果。
- 支持展示平均成绩、总学分、已获学分等数据。

### 3.9 选课时间窗口模块

- 管理员可配置选课窗口和退课窗口的开放时间。
- 支持设置开始时间、结束时间、窗口类型和启用状态。
- 用于控制学生是否可以在指定时间内进行选课相关操作。

### 3.10 消息中心模块

- 用于记录系统通知发送情况。
- 可查看接收对象、通知标题、发送状态和结果信息。
- 便于管理员掌握系统消息投递情况。

### 3.11 审计日志模块

- 记录管理员对关键数据的新增、修改、删除等操作。
- 支持按管理员、对象类型和操作内容进行查询。
- 方便后续追踪和责任定位。

### 3.12 学期总览与统计模块

- 提供课程数量、选课数量、待录成绩等统计信息。
- 帮助不同角色快速了解当前学期运行情况。
- 为系统管理和教学安排提供数据参考。

## 4. 模块关系说明

系统以“用户身份”为入口，不同角色进入系统后会看到不同的功能模块：

- 管理员侧重基础数据管理、流程控制和审计追踪。
- 教师侧重课程维护和成绩录入。
- 学生侧重课程选择、成绩查询和个人信息维护。

各模块之间通过课程、学生、教师、选课记录等基础数据形成关联，共同完成完整的选课业务流程。

## 5. 总结

本系统覆盖了学生选课业务中的主要环节，包括用户管理、课程管理、选课管理、成绩管理和过程控制等内容，能够较完整地支持高校课程选课场景下的日常教学管理需求。
